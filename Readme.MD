# E-Commerce Multi-Agent Support Bot 🤖

An AI-powered customer support system for Indian e-commerce 
built with LangGraph + LangChain + Groq (free LLM).

## Features
- 🌐 Hindi/Hinglish language support
- 😤 Emotion detection & tone adaptation  
- 🧠 Memory across sessions
- 🚚 Order tracking via Shiprocket
- 💳 Payment support via Razorpay

## Tech Stack
LangGraph | LangChain | Groq Llama 3 | SQLite | FastAPI | Streamlit
